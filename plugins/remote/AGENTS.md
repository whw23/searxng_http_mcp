# SearXNG HTTP MCP

This plugin provides web search capabilities via a SearXNG metasearch engine MCP server.

## Available MCP Tools

- **search** — Search the web across 200+ engines (Google, Bing, DuckDuckGo, etc.)
- **autocomplete** — Get search query suggestions
- **engine_info** — List available search engines and categories

## Usage

Use the `search` tool for web queries. Prefer `categories` over `engines` to narrow results. Use `pages=2-3` for comprehensive research and `format=full` for quality evaluation.

For complex research tasks, delegate to the `web-search-via-searxng` skill which implements a structured multi-round search workflow.
